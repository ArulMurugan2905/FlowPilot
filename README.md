# FlowPilot

## Computers remember files. FlowPilot remembers work.

FlowPilot is a personal work-memory agent built into Windows that helps users recover project context, decisions, notes, and unfinished tasks through natural language interactions.

Instead of spending time reopening tabs, locating files, and remembering where work stopped, users can simply ask:

> Continue my Microsoft Hackathon project.

FlowPilot restores project context and helps users resume meaningful work instantly.

---

## Problem

Professionals, students, developers, and researchers frequently lose productivity when returning to projects after days or weeks.

Common challenges include:

* Reopening browser tabs
* Locating project files
* Finding notes and documentation
* Remembering previous decisions
* Identifying unfinished tasks

Current systems remember files.

They do not remember work.

---

## Solution

FlowPilot introduces Project Memories.

Users explicitly start and end Project Memory sessions.

During a session, FlowPilot captures:

* Project context
* Notes
* Decisions
* Tasks
* Related files
* Relevant resources

Nothing is captured outside user-authorized sessions.

---

## How FlowPilot Works

1. User starts a Project Memory session.
2. FlowPilot captures project context.
3. Session ends when the user chooses.
4. Context is organized using Microsoft Work IQ.
5. Microsoft Copilot reasons over the stored context.
6. Users resume work through natural language commands.

---

## Microsoft Work IQ Integration

FlowPilot leverages Microsoft Work IQ to create structured project memory from user-authorized work sessions.

Work IQ helps organize:

* Context
* Decisions
* Relationships
* Tasks
* Historical activity

This allows Microsoft Copilot to reason over project history and provide grounded responses.
